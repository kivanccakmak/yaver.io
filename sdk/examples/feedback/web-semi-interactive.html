Semi Interactive Mode STREAMING

Agent sees your screen live and comments, but doesn't auto-fix. Conversation mode.

import { YaverFeedback } from '@yaver/feedback-web';

// Semi Interactive: streaming video + conversation, no auto-fix
YaverFeedback.init({
  trigger: 'keyboard', // Ctrl+Shift+F
  // feedbackMode is selected at runtime by the user
});

// In this mode:
// 1. Screen + mic stream to agent (same as full interactive)
// 2. Agent watches and comments on what it sees
// 3. Agent says: "I see an issue with the nav overlap"
// 4. User: "yeah fix that in the next release"
// 5. Agent: "Noted. Added to backlog."
// 6. NO hot reload — agent observes and converses only
// 7. User can say "fix it now" to switch to full interactive for that issue
// 8. Or "keep in mind" to defer to next session

Agent Commentary (simulated)
