How Full Interactive Mode Works

import { YaverFeedback } from '@yaver/feedback-web';

// Full Interactive: agent sees everything live, fixes instantly
YaverFeedback.init({
  trigger: 'floating-button',
  // No agentUrl needed — auto-discovers on LAN
});

// In this mode:
// 1. You click "Start Recording" → screen + mic stream to agent
// 2. Agent's vision model watches for UI bugs in real-time
// 3. You say "that button is broken" → agent hears it
// 4. Agent changes code → webpack HMR pushes fix → you see it instantly
// 5. Agent may proactively say: "I see a layout overlap, want me to fix it?"
// 6. Commentary level controls how chatty the agent is (0-10)
